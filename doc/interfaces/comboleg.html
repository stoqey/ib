Optional openClose

openClose: number
  • Defined in src/api/contract/comboLeg.ts:33

Specifies whether an order is an open or closing order. For institutional customers to determine if this order is to open or close a position.

  • 0 - Same as the parent security. This is the only option for retail customers.
  • 1 - Open. This value is only valid for institutional customers.
  • 2 - Close. This value is only valid for institutional customers.
  • 3 - Unknown.
